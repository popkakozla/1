# Контрольные вопросы

## 1. Что такое командная строка?

Командная строка — интерфейс взаимодействия пользователя с операционной системой через текстовые команды.

---

## 2. Основные команды управления файлами

- copy
- del
- ren
- md
- rd
- dir

---

## 3. Команды вывода информации системы

- systeminfo
- ver
- date
- time

---

## 4. Команды ввода/вывода файлов

- copy
- type
- more

---

## 5. Возможно ли полноценное управление системой?

Да, через командную строку можно управлять файлами, процессами, сетью и системными параметрами.

---

## 6. Отличие CMD Windows от MS-DOS

CMD работает поверх Windows и использует Windows API, а MS-DOS являлся отдельной операционной системой.

---
