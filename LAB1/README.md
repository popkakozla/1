
# Лабораторная работа №1

## Тема
Командная строка Windows

## Цель работы
Изучение основных команд командной строки Windows и получение практических навыков работы с файлами и каталогами.

## Выполненные задачи
- создание каталогов;
- создание файлов;
- копирование файлов;
- удаление файлов и папок;
- сортировка файлов;
- работа с prompt;
- вывод содержимого файлов.

## Используемые команды
- md
- rd
- copy
- del
- dir
- prompt
- cls
- tree

## Результат
Были освоены базовые команды командной строки Windows.
