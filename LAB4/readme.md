# Лабораторная работа №4

## Тема
Работа с системным реестром Windows

## Цель работы
Изучение структуры системного реестра Windows и получение практических навыков работы с редактором реестра Regedit.

## Теоретические сведения
Системный реестр Windows представляет собой централизованную базу данных, в которой хранятся настройки операционной системы, драйверов, программного обеспечения и учетных записей пользователей. Реестр имеет древовидную структуру и состоит из разделов, подразделов и параметров.

Основные корневые разделы реестра:
- HKEY_CLASSES_ROOT;
- HKEY_CURRENT_USER;
- HKEY_LOCAL_MACHINE;
- HKEY_USERS;
- HKEY_CURRENT_CONFIG.

Параметры реестра могут иметь различные типы данных:
- REG_SZ;
- REG_DWORD;
- REG_BINARY;
- REG_MULTI_SZ;
- REG_EXPAND_SZ.

## Ход выполнения работы
В ходе выполнения лабораторной работы был запущен редактор реестра Windows с помощью команды regedit. Были изучены основные разделы системного реестра и их структура.

В разделе HKEY_CURRENT_USER\Control Panel\Desktop были рассмотрены параметры, отвечающие за настройки рабочего стола Windows. Также был исследован подраздел WindowMetrics, содержащий параметры размеров иконок, шрифтов, рамок окон и других элементов интерфейса.

Дополнительно были изучены параметры:
- ShellIconSize;
- FontSmoothing;
- Wallpaper.

Далее был рассмотрен раздел HKEY_CLASSES_ROOT, отвечающий за ассоциации файлов и взаимодействие расширений файлов с приложениями.

## Вывод
В результате выполнения лабораторной работы были изучены структура системного реестра Windows, основные корневые разделы, параметры и типы данных реестра, а также получены навыки работы с редактором реестра Regedit.
