# Лабораторная работа №3

## Тема
Использование приемов работы с файловой системой NTFS. Назначение разрешений доступа к файлам и папкам.

## Цель работы
Изучение механизмов разграничения доступа в файловой системе NTFS.

## Выполненные задачи
- создание пользователей;
- настройка NTFS разрешений;
- наследование прав;
- изменение владельца;
- копирование и перемещение папок;
- анализ разрешений доступа.

## Результат
Были изучены механизмы разграничения доступа и наследования разрешений NTFS.
