# Лабораторная работа №2

## Тема
Управление памятью и процессами в ОС Windows

## Цель работы
Изучение процессов Windows и способов управления ими через диспетчер задач и командную строку.

## Выполненные задачи
- запуск процессов;
- просмотр процессов;
- завершение процессов;
- управление explorer.exe;
- использование tasklist и taskkill.

## Используемые команды
- tasklist
- taskkill
- start
- cd
- dir

## Результат
Были изучены механизмы работы процессов Windows и способы управления ими.
